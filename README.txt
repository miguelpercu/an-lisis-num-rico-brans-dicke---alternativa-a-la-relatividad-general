README for Supplementary Material

This supplementary material contains the Jupyter Notebook "Python Pipeline for Real Data Analysis.ipynb," which implements the numerical and statistical analysis of the Brans-Dicke (BD) vs. ΛCDM cosmological models using real data from the Pantheon Type Ia supernova catalog. The code reproduces the results presented in the manuscript "Evolución cosmológica en la teoría de Brans-Dicke: una aproximación numérica y análisis estadístico comparado con ΛCDM" (DJ14072).

Instructions:
1. Install dependencies: `pip install numpy scipy pandas matplotlib jupyter`.
2. Launch Jupyter Notebook: `jupyter notebook`.
3. Open and run "Python Pipeline for Real Data Analysis.ipynb".
4. Results (plots, tables) will be generated in the `results/` directory.
